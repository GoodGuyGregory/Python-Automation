Basic format for an exception 

```python 
#!/usr/bin/python3 

num1 = input ("Enter number") 

try:   print(num1*num1) 

except TypeError:   
   print("An TypeError occurred") 

else: 
   print("There is no exception")

print("Operation completed")
```
